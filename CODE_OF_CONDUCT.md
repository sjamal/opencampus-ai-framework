# Code of Conduct

The OpenCampus AI Framework is intended to be a welcoming and inclusive community.

We expect all contributors to:

- Be respectful.
- Assume positive intent.
- Provide constructive feedback.
- Encourage learning.
- Respect differing viewpoints.
- Focus discussions on improving the project.

Harassment, discrimination, or personal attacks are not acceptable.

This document will evolve as the project grows.
