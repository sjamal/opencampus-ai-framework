# Glossary

## Agent

An AI component capable of reasoning about a task and deciding which tools or actions to use.

---

## API

Application Programming Interface.

A defined way for software systems to communicate.

---

## MCP

Model Context Protocol.

An open protocol that enables AI applications to securely discover and use external tools and resources.

---

## RAG

Retrieval-Augmented Generation.

A technique that combines information retrieval with Large Language Models to generate responses grounded in external knowledge.

---

## Workflow

A defined sequence of business activities, approvals, and state transitions.

---

## ADR

Architecture Decision Record.

A document describing an important architectural decision and the rationale behind it.
